Para desenvolver no PodcastAds ou modificar este exato _Guia do Ouvinte_ (criando seus próprios arquivos Markdown em `apps/docs/content`), você deve baixar o repositório e prepará-lo.

## Requisitos de Sistema

- **Node.js**: Versão 18.x ou superior.
- **Gerenciador de Pacotes**: `pnpm` (recomendável na versão 9+).
- _Git_ ativo e clonado.

## Inicializando Tudo 🚦

Como utilizamos uma arquitetura em Monorepo via `Turborepo` conectando o `Next.js App Router`, os pacotes vitais (UI, Core) requerem os mesmos comandos.

### 1. Instaure as Dependências

Pela primeira vez que acessar o root (na mesma pasta do `package.json` matriz), baixe toda a internet:

```bash
pnpm install
```

### 2. Ligue os Motores (Dev Server)

Ao invés de tentar iniciar pacotes avulsos, execute a inteligência global para puxar o pacote do Portal+Docs:

```bash
pnpm dev:docs
```

> [!WARNING]
> É possível que a rotina dispare avisos sobre o NPM caso um arquivo `.npmrc` tente validar pacotes locais sem seu token de registro original. O ambiente rodará independentemente se o log der pequenas queixas sobre variáveis faltando.

### 3. Acesse e Crie!

Depois dessa rodada rápida, use seu navegador para acessar localmente a maravilha viva:

`http://localhost:3000`

Se quiser criar uma nova documentação? Vá aos subdiretórios `apps/docs/content/docs` e comece a escrever sua própria página `.mdx` utilizando nossos Componentes UI customizados!
