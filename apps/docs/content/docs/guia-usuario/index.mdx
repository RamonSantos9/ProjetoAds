Bem-vindo ao portal **PodcastAds**! Este guia foi criado para ajudar você a aproveitar ao máximo nossa plataforma de podcasts acadêmicos.

## O Que é o PodcastAds?

É um portal onde os alunos de ADS compartilham conhecimentos, entrevistas e projetos através de podcasts de alta qualidade. Aqui você encontra desde pautas técnicas até discussões sobre carreira e mercado.

## Navegando no Portal

### Página Inicial (Home)
Logo ao entrar, você verá o **Destaque do Dia** e os **Últimos Episódios**. 
- Role a página para ver o "Marquee" (texto deslizante) com os temos quentes.
- Use o menu superior para acessar rapidamente a biblioteca completa.

### Biblioteca de Episódios (`/episodios`)
Uma grade completa com todos os episódios já gravados. Cada "card" mostra:
- **Título e Categoria**: Do que se trata o episódio.
- **Participantes**: Quem foi entrevistado ou participou.
- **Duração**: Quanto tempo dura o conteúdo.
- **Status**: Se já está disponível para ouvir ou se é uma pauta futura.

## Painel do Ouvinte (`/dashboard`)

Se você for um aluno autenticado, terá acesso a um painel personalizado onde poderá:
- Ver estatísticas de consumo.
- Acompanhar o progresso de pautas em que você estiver envolvido.
- Acessar conteúdos exclusivos para membros da comunidade ADS.

---

> [!TIP]
> Você pode usar a barra de busca (ícone de lupa no topo) para encontrar episódios por título ou convidado em segundos!
