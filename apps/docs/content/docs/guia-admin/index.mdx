Bem-vindo à área de controle do **PodcastAds**! Este painel é restrito e permite gerenciar todos os aspectos do conteúdo, desde a criação de pautas até a publicação de episódios e análise de dados.

## O Que é a Área Admin?

Localizada em `/admin`, esta interface é o coração administrativo da plataforma. Ela foi desenhada para facilitar o fluxo de trabalho dos gestores de conteúdo, permitindo que foquem na qualidade das entrevistas enquanto a plataforma cuida da organização dos dados.

## Navegando no Painel Admin

Ao entrar, você verá o **Painel Geral (Home)** com um resumo das atividades:
- **Resumo de Estatísticas**: Total de episódios, convidados e visualizações.
- **Ações Rápidas**: Atalhos para criar novos episódios ou ver relatórios.

### Estúdio e Projetos (`/admin/estudio`)
Aqui você gerencia as pautas ("Projetos") que ainda não se tornaram episódios gravados.
- **Organização por Cards**: Use cards interativos com menus de ação rápida.
- **Fluxo de Trabalho**: Mantenha o controle do que está em roteirização, agendamento ou gravação.

### Gestão de Episódios (`/admin/episodios`)
A biblioteca central de tudo o que já foi produzido.
- **CRUD Completo**: Crie, Leia, Atualize ou Delete informações.
- **Edição Rica**: Altere capas, descrições e status com facilidade.

### Relatórios e BI (`/admin/relatorios`)
Acesse dados profundos sobre o desempenho do podcast.
- **Análise Quantitativa**: Gráficos de crescimento e engajamento.
- **Exportação**: Gere dados para apresentações acadêmicas.

---

<Callout type="warn">
Lembre-se que alterações feitas aqui são imediatas no Portal Público. Sempre revise os metadados antes de publicar um novo episódio.
